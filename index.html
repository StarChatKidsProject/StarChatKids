Getting Started

You can set up P-Chessbot in just a few steps:

  1. Clone the repo:
    git clone https://github.com/StarChatKidsProject/P-chessbot-Project.git
  2. Install dependencies:
    pip install -r requirements.txt
  3. Run the engine:
    python engine.py
  4. Connect it with your favorite chess GUI (Arena, CuteChess, PyChess).

Engine Rating

P-Chessbot is approximately rated at ~2000 Elo based on self-play and testing against other open-source engines.

This makes it ideal for intermediate-level chess players to practice and improve their skills.
