Documentation

Everything you need to get started with P-Chessbot.

Installation Guide

  1. git clone https://github.com/StarChatKidsProject/P-chessbot-Project.git
  2. cd P-chessbot-Project
  3. pip install -r requirements.txt
  4. python engine.py

Using the Engine

P-Chessbot follows the UCI (Universal Chess Interface) standard. You can use it with GUIs like Arena, PyChess, or CuteChess.

Configuration

Settings you can adjust:

  • Search depth
  • Time per move
  • Evaluation parameters
